# Base42 Website SolidStart Port

SolidStart recreation of the Base42 website, built in a separate folder so the original Next.js repo stays untouched.

## Commands

```bash
npm install
npm run dev
npm run build
npm start
```

## Notes

- Public assets were copied from the original project.
- Event data comes from the same Strapi instance at `https://cms.42.mk`.
- Video data uses the same YouTube API environment variables as the original site.
